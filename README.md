## 每章结构（Harness 8模块）

1. 【模块定位】—— 学科位置与考试权重
2. 【知识图谱】—— ASCII 树状结构图
3. 【核心概念】—— ★必考 + ☆重要概念表
4. 【重点公式/算法】—— 关键数字与公式
5. 【方法比较】—— 对比辨析表
6. 【考试聚焦】—— 考点分级与题型预测
7. 【模拟练习】—— 名词解释/简答/论述，折叠答案
8. 【自测清单】—— 掌握度自检

## 公式渲染

本项目使用 LaTeX 语法编写数学公式。GitHub Markdown 原生支持 `$...$`（行内）和 `$$...$$`（块级）渲染。

## 使用方式

1. 按章节顺序复习，每章通读8模块
2. 考前用附录「重点概念速查表」快速回顾（缩写表/数字汇总/核心对比/公式汇总）
3. 闭卷完成附录《综合练习题库》：选择/判断题侧重跨章节辨析，简答/论述题侧重答题结构（先列要点再展开），对照答案查漏补缺
4. 配合各章【模拟练习】巩固单章知识点，闭卷作答后对照答案找薄弱点
