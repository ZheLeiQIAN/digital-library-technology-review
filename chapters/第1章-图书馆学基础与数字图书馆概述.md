## 【核心概念】

### 1.1 图书馆的类型与职能 ★

**我国图书馆按性质和功能划分的类型：**

| 类型 | 定义与特点 |
|------|----------|
| 国家图书馆 | 政府设立的国家藏书中心，担负国家总书库职能；馆际互借中心、国际书刊交换中心、全国书目和图书馆学研究中心 |
| 公共图书馆 | 各级人民政府投资兴办、面向社会公众开放；分布广泛，是面向社会服务的主体 |
| 大学图书馆 | 高等院校的文献信息中心，办好大学的三大支柱之一；主要服务本校师生，逐步向社会开放 |
| 学校图书馆 | 大学以外的各类学校图书馆（小学、中学、职业学校等）；为教学和提高学生文化素质服务 |
| 专门图书馆 | 也称专业/科研/学术图书馆；以某个学科领域为主建立；特定学科领域重要的文献信息来源 |
| 其他图书馆 | 企业图书馆、单位内部图书馆、部队图书馆等；为某一单位服务，有的具有保密性 |

**图书馆的四大职能：**
1. 保存人类文化遗产
2. 开展社会教育
3. 传递科学情报
4. 开发智力资源

**图书馆的六大服务：**
- 外借服务、阅览服务、参考咨询服务
- 馆际互借服务、文献传递服务
- 读者教育与读者培训服务

### 1.2 图书分类法 ☆

**《中国图书馆分类法》（中图法，Chinese Library Classification, CLC）——5个基本部类、22个基本大类：**

| 基本部类 | 基本大类 |
|---------|----------|
| 马列主义、毛泽东思想 | A 马克思主义、列宁主义、毛泽东思想、邓小平理论 |
| 哲学 | B 哲学、宗教 |
| 社会科学 | C 社会科学总论；D 政治、法律；E 军事；F 经济；G 文化、科学、教育、体育；H 语言、文字；I 文学；J 艺术；K 历史、地理 |
| 自然科学 | N 自然科学总论；O 数理科学和化学；P 天文学、地球科学；Q 生物科学；R 医药、卫生；S 农业科学；T 工业技术；U 交通运输；V 航空、航天；X 环境科学、安全科学 |
| 综合性图书 | Z 综合性图书 |

**其他主要分类法：** 杜威十进分类法、美国国会图书馆分类法、国际十进分类法、冒号分类法

### 1.3 数字图书馆的定义 ★

| 学者/机构 | 年份 | 定义 |
|----------|------|------|
| 美国数字图书馆联盟(DLF) | 1998 | 一个拥有专业人员等相关人员的组织，对数字式资源进行挑选、组织，提供智能化存取、解译、传播，保持其完整性和永存性 |
| William Y. Arms | 2000 | 具有服务功能的整理过的信息收藏，信息以数字化格式存储并可通过网络存取 |
| 刘炜 | 2000 | 社会信息基础结构中信息资源的基本组织形式，满足分布式面向对象的信息查询需要 |
| 李培等 | 2004 | 对有价值的多媒体信息进行收集、组织和规范再加工，通过网络提供高速横向跨库连接的多媒体信息存取服务 |

**数字图书馆的特征：** 具备数字资源、网络服务和特色技术特征的信息资源和知识服务形式。

### 1.4 数字图书馆的基本功能 ★

1. **数字化信息的采集** —— 信息资源数字化建设是重要内容
2. **数字化信息的储存与管理** —— 存储在计算机、磁盘、光盘（磁带）中；服务模式从C/S到B/S
3. **数字化信息的访问与查询** —— 文本数据库查询、多模态资料查询
4. **数字化信息的传送与发布** —— 基于TCP/IP协议，多媒体信息需要较大带宽
5. **数字化信息的权限管理** —— 控制用户访问权限，保护知识产权

### 1.5 数字图书馆的社会功能 ☆

- 保护人类信息资源
- 开展社会教育
- 传递使用信息
- 提供文化休闲
- 开发智力资源
- 开展网络导航
- 通信功能与宣传功能

### 1.6 数字图书馆与图书馆自动化的关系 ★

**图书馆自动化：** 图书馆内部业务操作的计算机化（目录、索引等书目信息存储的计算机化，检索、借阅业务的计算机化），**不强调馆藏的计算机化**。

**数字图书馆：** 图书馆内部业务操作的计算机化 + 馆藏的数字化。

**发展阶段：**
1. 第一阶段：以单一图书馆计算机管理系统为标志的初级阶段
2. 第二阶段：以网络化服务为标志的电子文献服务阶段
3. 第三阶段：以数字图书馆为标志的高级阶段

**二者关系：**
1. 图书馆自动化是数字图书馆的基础之一（数字化书目信息来源于自动化系统）
2. 实现图书馆自动化是建设数字图书馆的必经阶段（但不是简单扩展）
3. 图书馆自动化是在传统图书馆理论框架下应用计算机技术；数字图书馆是在理论与技术上超越传统图书馆的新发展

## 【方法比较】

| 维度 | 传统图书馆 | 数字图书馆 |
|------|-----------|------------|
| 存储介质 | 纸质文献 | 数字化格式 |
| 服务方式 | 到馆服务为主 | 网络远程服务 |
| 检索方式 | 手工/卡片检索 | 计算机检索 |
| 资源共享 | 馆际互借受限 | 高速横向跨库连接 |
| 服务时间 | 受开馆时间限制 | 无时间限制 |
| 空间占用 | 馆舍空间压力大 | 存储空间小 |

## 【模拟练习】

<details>
<summary>【练习1】单选题：数字图书馆的基本功能不包括以下哪项？</summary>

A. 数字化信息的采集  
B. 数字化信息的储存与管理  
C. 纸质图书的装订修复  
D. 数字化信息的权限管理

**答案：C**

**解析：** 数字图书馆的5大基本功能包括：数字化信息的采集、储存与管理、访问与查询、传送与发布、权限管理。纸质图书的装订修复属于传统图书馆的物理维护工作，不是数字图书馆的基本功能。

</details>

<details>
<summary>【练习2】多选题：以下哪些属于我国图书馆按性质和功能划分的类型？</summary>

A. 国家图书馆  
B. 公共图书馆  
C. 大学图书馆  
D. 专门图书馆  
E. 社区图书馆

**答案：A、B、C、D**

**解析：** 我国图书馆按性质和功能划分为6种类型：国家图书馆、公共图书馆、大学图书馆、学校图书馆、专门图书馆、其他图书馆（企业图书馆、单位内部图书馆、部队图书馆）。"社区图书馆"属于公共图书馆的服务延伸，不是独立类型。

</details>

<details>
<summary>【练习3】简答题：简述数字图书馆与图书馆自动化的主要区别。</summary>

**参考答案：**

1. **范围不同：** 图书馆自动化是图书馆内部业务操作的计算机化，不强调馆藏的计算机化；数字图书馆不仅包括业务操作计算机化，还包括馆藏的数字化。

2. **发展阶段不同：** 图书馆自动化是初级阶段，数字图书馆是高级阶段。

3. **理论框架不同：** 图书馆自动化是在传统图书馆理论框架下应用计算机技术来改善服务与管理；数字图书馆是在理论与技术上超越传统图书馆的新发展。

4. **关系：** 图书馆自动化是数字图书馆的基础之一，实现自动化是建设数字图书馆的必经阶段，但数字图书馆不是自动化系统的简单扩展。

</details>

<details>
<summary>【练习4】填空题：《中国图书馆分类法》将图书分为___个基本部类，___个基本大类。</summary>

**答案：5；22**

**解析：** 中图法分为5个基本部类（马列主义、哲学、社会科学、自然科学、综合性图书）和22个基本大类（A-Z，其中I、O等为大类）。

</details>

</details>
